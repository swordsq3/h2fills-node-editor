# 키보드 단축키 & 마우스 조작

## 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl + Z` | 실행 취소 (Undo) |
| `Ctrl + Y` | 다시 실행 (Redo) |
| `Ctrl + S` | 그래프 저장 (JSON) |
| `Ctrl + O` | 그래프 불러오기 |
| `Ctrl + N` | 새 그래프 |
| `Ctrl + Space` | 블록 빠른 추가 (Quick Insert) |
| `Delete` | 선택된 노드·링크 삭제 |

## 마우스 조작

| 조작 | 동작 |
|------|------|
| 좌클릭 드래그 (빈 공간) | 박스 선택 |
| 좌클릭 드래그 (노드) | 노드 이동 |
| 중간 버튼 드래그 | 캔버스 이동 (Pan) |
| `Ctrl + 스크롤` | 캔버스 확대/축소 (Zoom) |
| 포트 → 포트 드래그 | 링크 연결 |
| 링크 클릭 후 `Delete` | 링크 삭제 |
| 노드 더블클릭 (Scope) | 스코프 상세 팝업 열기 |
| 팔레트 버튼 클릭 | 캔버스 중앙에 노드 추가 |
| 팔레트 아이콘 드래그 → 캔버스 | 드롭 위치에 노드 추가 |

## 툴바 버튼

| 버튼 | 색상 | 기능 |
|------|------|------|
| **Run** | 녹색 | 시뮬레이션 시작 (검증 후) |
| **Pause** | 주황 | 일시 정지 |
| **Step** | 파랑 | 1 틱 실행 |
| **Reset** | 회색 | 시뮬레이션 초기화 |
| **↩** | 보라 | Undo |
| **↪** | 보라 | Redo |

## 메뉴

| 메뉴 > 항목 | 기능 |
|------------|------|
| 파일 > 새로 만들기 | 그래프 초기화 |
| 파일 > 저장 | JSON 저장 |
| 파일 > 불러오기 | JSON 열기 |
| 파일 > 템플릿 불러오기 | 예제 그래프 로드 |
| 파일 > 최근 파일 | 최근 8개 파일 |
| 파일 > 모델 정보 | 제목·작성자·설명 편집 |
| 파일 > 결과 내보내기 (CSV) | 스코프 데이터 CSV 저장 |
| 편집 > 실행 취소 | Ctrl+Z |
| 편집 > 다시 실행 | Ctrl+Y |
| 편집 > 블록 빠른 추가 | Ctrl+Space |
| 시뮬레이션 > Run / Pause / Step / Reset | 시뮬레이션 제어 |
| 보기 > 스코프 창 | 이벤트 로그 창 열기 |

## Quick Insert 패널 (Ctrl+Space)

마우스 위치에 팝업이 열립니다:

1. `Ctrl + Space` 로 패널 열기
2. 검색창에 블록 이름 입력 (예: "valve", "탱크")
3. 목록에서 클릭 → 현재 마우스 위치에 블록 추가
4. 패널 자동으로 닫힘

## 상태바 (하단)

```
[상태 메시지]  Nodes: 4  Links: 7  Undo: 3  |  단축키 안내
```

| 항목 | 의미 |
|------|------|
| Nodes: N | 현재 캔버스의 노드 수 |
| Links: N | 현재 캔버스의 연결 수 |
| Undo: N | Undo 스택 깊이 |
